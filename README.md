## **React-RSVP**

React-RSVP is web application for inviting guests. Every guest can confirm her/his is coming. Counter calculate number of total, confirmed and unconfirmed guests.



### **TECHNOLOGIES**

HTML5, CSS3, JavaScript (ES6), React, Npm
